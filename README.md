# Halal Food Maps

A visual mockup for a map-based halal restaurant discovery page, centered around Shibuya, Tokyo.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Leaflet + react-leaflet

## Run

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Features

- **Map**: Centered on Shibuya Station with 8 dummy halal-friendly restaurants
- **Markers**: Shape = category (Ramen, Sushi, Yakiniku, Cafe), Color = halal level
- **Filters**: Category, halal level, prayer room toggle
- **Info panel**: Click a marker to see restaurant details below the map
- **Legend**: Floating legend explaining marker shapes and colors

## Design

Clean, trustworthy, modern UI optimized for Muslim travelers. Desktop-first, responsive.
